# 🚀 AI Lab — Fast, Hacky Experiments with LLMs

Welcome to my **AI experimentation playground** 🤖

This repository tracks **projects generated with AI tools** such as **Qwen Code and other LLM-based assistants**, along with the **human fixes, feedback, hacks, and lessons learned** while turning those outputs into something that actually works.

The philosophy is simple:

⚡ move fast  
🧪 experiment constantly  
🛠 fix what the AI breaks  
🧠 document what was learned  
📚 spend the saved time **reading books and taking photographs**

AI writes the first draft.  
Humans make it **real**.

---

# 🎯 Why this repo exists

Modern AI tools can generate **huge amounts of code quickly**, but they still need:

- debugging 🔧  
- architectural decisions 🧱  
- security sanity checks 🔐  
- real-world testing 🧪  

This repo captures **that interaction layer between AI and human engineering**.

Think of it as:

> 🧠 *a memory bank of experiments with AI*

Each project documents:

- what the AI generated
- what worked
- what broke
- what had to be fixed
- what we learned from it
